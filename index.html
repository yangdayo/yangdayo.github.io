Java预科

发表于 2023-01-21

java预科

Java诞生

c&c++

1972年c诞生

  • 优点：贴近硬件，运行极快，效率极高
  • 缺点：指针与内存管理

1982年c++诞生

  • 面向对象

  • 兼容c

  • 运用于图形领域，游戏等

    反抗：新的语言的需求

  • 语法有点像c

  • 没有指针

  • 没有内存管理

  • 真正的可移植性，编写一次，到处运行

  • 面向对象

  • 类型安全

  • 高质量的类库

java初生

  • 1995年的网页简单粗糙，缺少互动性

  • 诞生图形界面的程序Applet

  • J2SE：去占领桌面

  • J2ME：去占领手机

  • J2EE：去占领服务器

    java发展

  • 基于java开发了巨多平台，系统，工具

  • 构建工具：Ant,Maven,Jekins

  • 应用服务器：Tomcat,Jetty,Jboss,Websphere,weblogic

  • WEB开发：Spring,myBatis,Hibernate,Struts

  • 开发工具：Eclipse,Netbean,Intellij idea,Jbuilder

  • 。。。。。。。

  • 2006：hadoop

  • 2008：Android

java特性和优势

  • 简单性

  • 面向对象

  • 可移植性

  • 高性能

  • 分布性

  • 动态性

  • 多线程

  • 安全性

  • 健壮性

    JDK,JRE,JVM

    JDK:java development kit (java开发者工具)

    JRE:java runtime environment (java运行时环境)

    JVM:java virtual machine (java虚拟机)

ps:

  • JDK包含了JRE，在JRE基础之上扩充了一些开发工具
  • JVM实现了在各个系统系都能运行java，真正实现write once,run anywhere!
  • 所以接下来安装JDK开始java的开发学习吧！

DOS简介

DOS，是磁盘操作系统（英文：Disk Operating System）的缩写，是个人计算机上的一类操作系统。Microsoft Windows版本是以DOS为基础。

常用DOS命令

在DOS窗口也可以打开应用，如calc打开计算器，mspaint打开画图本，notepad打开记事本

同时在DOS窗口输入文件名等，可通过Tab自动补全
